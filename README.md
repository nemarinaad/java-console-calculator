# Java Console Calculator

This is a simple console calculator written in Java. 
It allows the user to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

Features:
- Loops until the user chooses to exit
- Validates the operator input (+, -, *, /)
- Handles division by zero
- Uses Scanner for console input

Created as a practice project while learning Java programming. 
This project demonstrates basic programming concepts like loops, conditionals, and user input handling.
